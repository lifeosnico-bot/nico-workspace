## Bot Map (always the same)
- **Nico** = the agent (same ID as Memo)
- **Memo** = legacy name for Nico (same agent)
- **LettaBot** = Telegram bridge only (not an agent)
- **Claude Code** = interactive dev assistant (separate)

## If Telegram is working
1) Send: `ping`
2) If it replies, you are connected to Nico on the M1.

## If Telegram fails and you can reach the M1
1) Open Terminal on the M1.
2) Run:

```
pkill -f "lettabot server" || true
pkill -f "lettabot/dist/main.js" || true
pkill -f "letta-ai/letta-code/lettat.js" || true
launchctl bootstrap gui/$(id -u) "/Users/lifeos.nico/Library/LaunchAgents/com.nico.lettabot.plist" 2>/dev/null || true
launchctl kickstart -k gui/$(id -u)/com.nico.lettabot
pgrep -fl "lettabot"
```

3) Send `ping` again in Telegram.

## If you cannot reach the M1
1) Open Claude iOS app → Claude Code → start a remote session (cloud).
2) Or use Safari: https://claude.ai/code
3) Letta web chat is cloud-only (no M1 access).
